UIImage(imageNamed586h) Category Reference

Declared in UIImage+imageNamed586h.h

Overview

imageNamed586h is a category of UIImage, provides you a method to display 586h images.

UIKit uses the -568h suffix to load a different image on the iPhone 5 only for the Default.png. This category adds a method to use in replacement for [UIImage imageNamed:] that searches for the file with the suffix for any image.
